# Kirim pesan WhatsApp via API
curl -X POST https://api.wahagateway.com/v1/sessions/{id}/messages/text
  -H "X-Api-Key: API_KEY_ANDA"
  -H "Content-Type: application/json"
  -d '{
    "to": "62812345xxxxx",
    "device_id": "device_id_anda",
    "message_type": "document",
    "document": "https://example.com/document.pdf",
    "filename": "document.pdf",
    "caption": "Dokumen contoh"
  }'

Fitur Unggulan

Semua yang Anda butuhkan untuk mengintegrasikan WhatsApp ke aplikasi Anda

API Ramah Developer

RESTful API yang dirancang untuk developer. Dokumentasi lengkap, contoh kode, dan SDK dalam berbagai bahasa pemrograman.

Mendukung Berbagai Jenis Pesan

Kirim berbagai jenis pesan: Text, Gambar, Video, dan Dokumen. Dukungan lengkap untuk semua format media dengan caption dan metadata.

Webhook Real-Time

Terima notifikasi instan via webhook saat pesan masuk. Konfigurasi webhook mudah, dukungan multiple endpoint, dan retry mechanism untuk keandalan tinggi.

Template Management

Buat dan kelola template pesan dengan variabel dinamis. Sempurna untuk pesan otomatis, notifikasi, dan kampanye marketing.

OTP Management

Sistem One Time Password (OTP) terintegrasi dengan template. Generate, kirim, dan verifikasi kode OTP dengan mudah.

Analitik & Monitoring

Lacak penggunaan API, tingkat pengiriman pesan, dan metrik performa. Buat keputusan berbasis data.

Multi Device

Kelola beberapa device WhatsApp secara bersamaan. Sempurna untuk bisnis dengan banyak akun atau departemen.

Personal & Group Chat

Kirim pesan ke personal chat atau grup WhatsApp. Dukungan lengkap untuk berbagai jenis pesan ke semua tipe chat.
